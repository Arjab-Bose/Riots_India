This repository contains data analysis and visualizations of riot events across India from 2016 to present. The project aims to track temporal trends and spatial distribution of civil and religious unrest using empirical data.

Data Source: ACLED (Armed Conflict Location & Event Data Project)

Riots spike up before Lok Sabha elections of 2019 and 2024. West Bengal (WB), Jammu & Kashmir (JK), and Uttar Pradesh (UP) exhibit the highest volume of such events.
